Every step matters. Skipping one usually breaks later steps.

### Copy Code Exactly
The code in the guide is tested. Don't try to "improve" it while learning.

### Keep Both Running
Backend and Frontend must both run simultaneously on different ports.

Both have `.env` files. These must be configured correctly.

### Virtual Environment is Critical
Always activate Python venv before running anything. (.\venv\Scripts\Activate.ps1)

## 🐛 WHEN THINGS BREAK

### Most Common Issues
1. **"Module not found"** → Activate venv, run pip install
2. **"Port already in use"** → Kill process, use different port
3. **"CORS errors"** → Check .env files, ensure both services running
4. **"Connection refused"** → Backend not running, check terminal 1
5. **"Can't find localhost:3000"** → Frontend not running, check terminal 2

**All fixed?** → Go back to QUICK_START.md → "Common Issues & Fixes"
